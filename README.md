# Shi Ting (Winnie) Lin - Professional Resume

This is a high-performance, precision-engineered professional resume built with Next.js, Tailwind CSS, and optimized for A4 PDF export.

## Key Features
- **2026 Digital Industrial Aesthetic**: Modern, high-contrast design with a focus on information density and visual impact.
- **Precision A4 Print Engine**: Custom CSS ensuring 1:1 pixel-perfect PDF export without margins or layout shifts across browsers.
- **Responsive Layout**: Seamless transition from mobile preview to desktop A4 container.
- **Auto-Deployment**: Integrated GitHub Actions for static hosting on GitHub Pages.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Components**: Radix UI / Lucide Icons
- **Deployment**: GitHub Actions / Pages

## Development
To start local development:
```bash
npm install
npm run dev
```

The app will be available at `http://localhost:9002`.
