# **App Name**: Winnie Lin Resume

## Core Features:

- Profile Display: Showcase professional profile, contact details, and a brief introduction.
- Skills & Tech Stack: Display technical skills, languages, frameworks, and domain knowledge.
- Experience Timeline: Present work experiences in a timeline format, detailing roles, responsibilities, and achievements.
- Education Summary: Provide a summary of educational background, including degrees and institutions.
- PDF Export: Enable users to download the resume as a PDF file.
- Animated Transitions: Use smooth transitions and animations to improve user engagement and overall site experience.

## Style Guidelines:

- Primary color: Deep teal (#008080) to convey professionalism and technical expertise.
- Background color: Light grayish-blue (#E0EEEE) for a clean and modern backdrop.
- Accent color: Soft gold (#B8860B) to highlight key information and CTAs.
- Body and headline font: 'Inter', a grotesque-style sans-serif, is recommended for both body and headline text because of its neutral and modern look.
- Employ a clean, single-page layout with clear sections for easy navigation.
- Incorporate subtle animations for transitions between sections.
- Use professional and minimalistic icons to represent skills and experiences.
